# ☀️ 태양광 기반 스마트 식물 관리 시스템

> 임베디드 소프트웨어학과 졸업작품  
> Solar-Powered Smart Plant Management System

---

## 📌 프로젝트 개요

태양광 패널과 추적 시스템을 통해 발전 효율을 극대화하고, 수집한 에너지로 식물의 자동 급수 및 온도 관리를 수행하는 자립형 스마트 식물 관리 시스템.

---

---

## ⚙️ 주요 기능

### 1. 태양광 추적 시스템 (Solar Tracker)
- LDR 센서 4개로 태양 방향 감지
- 서보모터 2개로 패널 상하·좌우 각도 자동 조절
- 고정 패널 대비 발전 효율 향상

### 2. 자동 급수 시스템
- 토양 수분 센서로 수분 상태 측정
- 설정값 이하로 떨어지면 수중 펌프 자동 작동
- 과급수 방지 로직 포함

### 3. 온도 관리 시스템
- DHT22 센서로 온습도 실시간 측정
- 설정 온도 초과 시 냉각 팬 자동 가동
- 사용자가 목표 온도 설정 가능

### 4. 실시간 모니터링 대시보드
- 라즈베리파이 기반 웹 서버 (Flask / Node.js)
- 발전량, 토양 수분, 온습도 실시간 그래프
- 히스토리 데이터 저장 및 조회

## 🎯 기대 효과

- 태양광 추적으로 고정 패널 대비 **발전 효율 20~30% 향상**
- 자동화된 식물 관리로 **인력 개입 최소화**
- 외부 전력 없이 동작하는 **완전 자립형 시스템**
- 실시간 모니터링을 통한 **데이터 기반 식물 관리**
